This project presents an executive-level sales dashboard built using the Superstore dataset. The objective was to provide a clear, decision-focused view of revenue performance, customer segmentation, and profitability.

Key Features
KPI summary for revenue, profit, and total orders
Revenue trend over time to track growth patterns
Sales breakdown by customer segment
Top customers by revenue with profitability context
Scatter plot analyzing revenue vs profitability
Interactive filters for region, segment, and category
Data Model

The dataset follows a transactional structure where each row represents an individual order line. The analysis simulates a star schema approach, treating orders as the fact table and customer/product attributes as dimensions.

Key Insights
Revenue is concentrated among a relatively small group of customers
The Consumer segment drives the largest share of total sales
Some high-revenue customers generate low or negative profit
Revenue trends show overall growth with periodic volatility

Tools Used:
